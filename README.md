Dieses Portfolio wurde von Grund auf mit HTML, CSS, JavaScript, Tailwind CSS
und Node.js gebaut — Schritt für Schritt, um meine Kenntnisse aufzufrischen

## 🚀 Lokale Installation

### Voraussetzungen
- Node.js >= 18
- npm >= 9

### Setup
```bash
# 1. Repository klonen
git clone https://github.com/so77nava/Portfolio.git
cd Portfolio

# 2. Abhängigkeiten installieren
npm install

# 3. Tailwind CSS bauen
npm run build:css

# 4. Server starten
npm run dev
```

Dann öffne `http://localhost:3000` im Browser.
