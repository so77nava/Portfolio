WebChat — Vollständige Chat-Anwendung

FAU Erlangen-Nürnberg · Web-basierte Systeme WS 2025/2026

Vollständige Chat-Anwendung, schrittweise über ein Semester entwickelt — von einem einfachen JavaScript-Client bis hin zu einer produktionsreifen, cloud-deployed Architektur mit Echtzeit-Kommunikation, KI-Features und sicherer Peer-to-Peer-Verbindung.

  • Echtzeit-Chat via WebSockets mit Typing Notifications und Desktop-Benachrichtigungen
  • Secret Messaging Modus via WebRTC (PeerJS) — direkte P2P-Verbindung ohne Server
  • Rechtschreibprüfung (TypeScript) & Wortvorhersage via WebAssembly (Rust/Markov)
  • REST API + WebSocket-Server mit Node.js, Express & MongoDB (persistente Datenhaltung)
  • Deployment in OpenStack (RRZE Cloud) mit Docker, nginx Reverse Proxy & docker-compose
  • Giphy API Integration & JavaScript-Bundling mit Webpack
